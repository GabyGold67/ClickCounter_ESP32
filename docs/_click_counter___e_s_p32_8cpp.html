Detailed Description

Code file for the ClickCounter_Esp32 library.

This libray's ClickCounter class models counters and tally counters in all it's functionality, and adds an extensive set of services and options to manage more complex and demanding counting applications. The displaying services are optionally provided through the use of a SevenSegDisplays library instantiated object.
